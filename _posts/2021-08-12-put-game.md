## Update game request (methode : PUT)
Required Role :
* ROLE_SUPERADMIN
* ROLE_OPERATOR
* ROLE_RESELLER

** one role is required.

To update a game you need to use endpoint /games.
Example :
```
http://127.0.0.1:8000/games/1
```
